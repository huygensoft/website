Common Questions

Do I need an Enterprise license?

While basic integrations work with Pro, an 8SC+ Enterprise license is required for the full 3CX Call Control API functionality.

Is Linux deployment supported?

Yes, we support both Windows and Linux based 3CX instances, using cross-platform .NET or Node.js stacks.

Can call automation happen alongside agent activity?

Absolutely. Our API is designed to support concurrent activity, allowing automated logic to run without interrupting your agents' web clients or handsets.
